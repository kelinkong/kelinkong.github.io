举例：

compareTo方法返回一个整数，表示两个字符串的字典顺序比较结果。它的返回值有以下含义：

  • 返回负数：如果调用compareTo方法的字符串（即b + a）在字典顺序上小于参数字符串（即a + b)，则返回一个负数。
  • 返回0：如果调用compareTo方法的字符串（即b + a）在字典顺序上等于参数字符串（即a + b)，则返回0。
  • 返回正数：如果调用compareTo方法的字符串（即b + a）在字典顺序上大于参数字符串（即a + b)，则返回一个正数。
